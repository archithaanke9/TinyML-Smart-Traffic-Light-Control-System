# TinyML Smart Traffic Light Control System

## Team Members

* A.ARCHITHA
* JYOTHI VERMA
* SURBEESHWAR DASH

## Project Overview

This project uses ESP32-CAM and TinyML to classify traffic density (Low, Medium, High) and dynamically control traffic signal timing.

## Technologies Used

* ESP32-CAM
* ESP32 DevKit
* Edge Impulse
* TensorFlow Lite for Microcontrollers
* Arduino IDE

## Features

* Real-time traffic image capture
* Vehicle density classification
* Dynamic traffic signal timing
* Edge AI (TinyML)

## Repository Structure

* Source_Code/ → Arduino source files
* Dataset/ → Training images
* Report/ → Project documentation
* Images/ → Architecture and output screenshots
